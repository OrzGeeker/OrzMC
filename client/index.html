Python 命令行启动器 - OrzPythonMC

Mac/Linux系统可以使用：代码仓库 - PyPI发布

安装方法:

# 安装应用到用户目录下
python3 -m pip install --user orzmc
# 添加用户二进制目录到环境变量
echo 'PATH=$PATH:'$(python3 -m site --user-base)/bin >> ~/.bashrc
# 使用添加的环境变量立即生效
source ~/.bashrc 
# 运行 orzmc 命令
orzmc
