Mac/Linux系统下还可以使用命令行工具 [**OrzMC**](https://pypi.org/project/OrzMC/) 进行游戏。

目前由于 `pip search` 服务不可用，且python2.x已经废弃，所以使用Python3.x从PyPI上安装**OrzMC**可以使用下面的方式：

```bash
python3 -m pip install pip_search && python3 -m pip install orzmc
```
